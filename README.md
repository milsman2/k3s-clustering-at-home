  Node 1 - 4 GB RAM // 128GB SATA III SSD

  Node 5 - 8 GB RAM // 128GB SATA III SSD

  Node 7 - 8 GB RAM // 128GB SATA III SSD
  Node 8 - 8 GB RAM // 128GB SATA III SSD
